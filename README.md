# ⬡ AXWELL PRO — Analista Sniper v4.0

> Plataforma de análise quantitativa para opções binárias em tempo real.  
> Desenvolvido para **Royal Capital** | Powered by Axwell Soluções Binárias

---

## 🚀 Funcionalidades

| Feature | Descrição |
|---|---|
| 🎯 **Sniper Board** | Score de confluência 0–100 com 6 indicadores |
| 📊 **Gráfico Avançado** | Candles + EMAs + Bollinger + MACD + RSI |
| 📈 **Performance** | Curva de patrimônio, P&L, distribuição W/L |
| 🛡️ **Gestão de Risco** | Kelly Criterion, Stop Loss/Win, Simulador Martingale |

## 📡 Indicadores Técnicos

- **RSI (14)** — Sobrecomprado/Sobrevendido
- **EMA 8 / 20 / 50** — Tendência multi-período  
- **MACD (12/26/9)** — Momentum e cruzamentos
- **Bollinger Bands (20, 2σ)** — Volatilidade e reversão
- **Stochastic (14/3)** — Confirmação de entrada
- **ATR (14)** — Medição de volatilidade

## 💻 Rodar Localmente

```bash
git clone https://github.com/axwelladm-boop/solu-oesbinarias.git
cd solu-oesbinarias
pip install -r requirements.txt
streamlit run axwell_pro.py
```

## ☁️ Deploy no Streamlit Cloud

1. Acesse [share.streamlit.io](https://share.streamlit.io)
2. Conecte este repositório
3. Main file: `axwell_pro.py`
4. Clique em **Deploy**
